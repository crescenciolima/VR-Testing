# Test Case – Usability

## Objective

Verify that the basic navigation and interaction controls in the metaverse are intuitive and easily learned by new users, ensuring a frustration-free initial experience.

---

## Steps

1. Access the metaverse through a web browser (without any prior tutorial) and observe the initial screen.
2. Attempt to perform the following actions without consulting any help resources:
   - Move around the environment (using WASD keys or arrow keys).
   - Interact with an NPC (Maria Felipa) to start a dialogue.
   - Access the progress menu to check completed quizzes.
3. Record the time spent and any difficulties encountered while performing each action.

---

## Expected Result

- The user quickly identifies the movement controls (e.g., through a subtle visual hint or standard gaming conventions).
- NPC interaction is triggered in a clear and understandable way (e.g., a hand icon or an “Press E” prompt).
- The progress menu is accessible through an intuitive button (e.g., a book icon or the “Tab” key).
- All actions are completed within 2 minutes without the need for external assistance.

---

## Success Metric

- 90% of tested users are able to complete all actions within the expected time and without critical errors.

---

## Notes

- Test on at least three platforms (web browser, mobile, and VR) to compare usability consistency across devices.
